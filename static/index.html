What’s Here?

Real-Time Upper Atmosphere Weather

This is a map detailing upper-atmosphere weather observations made by automated recording equipment aboard commercial aircraft in the Seattle area. It’s interactive; hover the mouse pointer over the colored dots that represent observations to see the details of that observation.

Mobile Device Support

Alas, it is quite limited at this time. Some mobile devices work with the above page, and some work with this page (touch the dots to bring up a dialog that gives an abbreviated version of the weather observation), but most don’t work properly with either page. A better solution that supports a wider range of devices is planned.
