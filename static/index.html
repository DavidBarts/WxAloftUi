What’s Here?

Real-Time Upper Atmosphere Weather

This is a map detailing upper-atmosphere weather observations made by automated recording equipment aboard commercial aircraft in the Seattle area. It’s interactive; hover the mouse pointer over the colored dots that represent observations to see the details of that observation.

Mobile-Friendly Version of the Above

If you’re using a smartphone, iPad, or other mobile device, this is probably the version you want. Instead of hovering with a mouse pointer (which you probably don’t have), touch the dots. Because of screen-size limitations, it displays less information per observation than the former page.
